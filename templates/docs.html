Search Campaigns API Documentation

GET

/api/campaigns/active

Returns all active campaigns with their request and contact counts

Response Example:

{
  "campaigns": [
    {
      "id": 1,
      "name": "Campaign Name",
      "status": "active",
      "total_requests": 10,
      "total_contacts": 5
    }
  ]
}
GET

/api/campaign/{campaign_name}/requests

Returns all requests for a specific campaign by name

Parameters:

campaign_name - Name of the campaign

Response Example:

{
  "requests": [
    {
      "id": 1,
      "campaign_id": 1,
      "req_text": "search phrase",
      "status": "pending"
    }
  ]
}
POST

/api/request/{request_id}/status

Updates the status of a specific request

Parameters:

  • request_id - ID of the request
  • status - New status ('inuse' or 'completed')

Response Example:

{
  "status": "Request status updated successfully"
}
POST

/api/contacts

Saves a new contact for a campaign

Form Parameters:

  • campaign_id - Campaign ID
  • request_id - Request ID
  • business_name - Name of the business
  • review_count - Number of reviews
  • phone - Phone number (optional)
  • domain - Website domain (optional)
  • email - Email address (optional)

Response Example:

{
  "status": "Contact saved successfully",
  "contact_id": 1
}
